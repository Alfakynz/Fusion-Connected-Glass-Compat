# 🔗 Fusion Connected Glass Compat

Fusion Connected Glass Compat (FCG Compat) adds compatibility between the Fusion Connected Glass and other mods.

## ✨ Compatibility

- [Aether](https://modrinth.com/project/YhmgMVyu)
- [Deeper and Darker](https://modrinth.com/project/fnAffV0n)
- [Wilder Wild](https://modrinth.com/project/AtHRJSUW)

## 📦 Installation

You can download Fusion Connected Glass Compat on [Modrinth](https://modrinth.com/project/FVonl8G5) or [CurseForge](https://www.curseforge.com/minecraft/texture-packs/fcg-compat)

## ⚙️ Requirements

You also need the following mod and resource pack for it to work properly:

- [Fusion](https://modrinth.com/project/p19vrgc2)
